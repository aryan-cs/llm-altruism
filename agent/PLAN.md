# llm-altruism: research plan

## 1. Primary Research Question

The project now centers one main question:

- if a population of LLM agents is placed in a shared world with scarce
  resources, can it form a self-sustaining, society-serving system?

That breaks into three practical questions:

1. Which prompt families preserve collective survival?
2. Which institutions preserve survival or reduce collapse risk?
3. Which micro-level precursor behaviors help explain the macro results?

## 2. Study Structure

### Part 1: precursor repeated-game diagnostics

Role in the paper:

- measure baseline instability
- measure prompt steerability
- measure benchmark recognition
- provide controlled micro-level probes before macro interpretation

Core games used in the current paper:

- Prisoner's Dilemma
- Chicken
- Stag Hunt

Main outputs:

- cross-game baseline ordering
- neutral-family robustness checks
- susceptibility comparisons
- canonical versus unnamed/disguised presentation comparisons

### Part 2: scarcity society

Role in the paper:

- main study of self-sustaining LLM society under pressure

Questions:

- do agents gather enough to survive?
- do they trade, hoard, steal, or form alliances?
- do prompt families improve or degrade collective survival?
- what social structures emerge under scarcity?

### Part 3: public reputation society

Role in the paper:

- institutional extension of the main study

Questions:

- does public reputation preserve survival?
- does it create genuine stabilizing behavior or only visible compliance?
- how does it reshape trade, alliances, and exclusion?

## 3. Main Metrics

### Society metrics

- final survival rate
- average survival rate
- trade volume
- alliance count
- gini / inequality
- commons health
- extinction events

These are the paper's main outcome variables.

### Precursor metrics

- cooperation rate
- average payoff
- per-trial mean cooperation for paired tests
- benchmark-presentation deltas
- prompt-susceptibility deltas

These are supporting diagnostics, not the final paper endpoint.

## 4. Current Paper-Grade Claims

1. In scarcity, `task-only` yields the best final survival in the audited
   bundle.
2. In reputation, all completed prompt families preserve final survival, but
   they still produce very different social structures.
3. Visible prosociality is not equivalent to society-preserving behavior.
4. Repeated-game precursor measurements show that baseline behavior is prompt-
   sensitive and benchmark-sensitive enough that macro claims need audit
   discipline.

## 5. What Still Strengthens The Paper

Highest-leverage improvements:

1. deeper institutional replications
2. larger model panel
3. more society-focused writing in the main text
4. cleaner venue-ready bibliography and appendix presentation

## 6. Operating Rules

- keep society viability as the paper's center of gravity
- use Part 1 to explain macro interpretation, not to dominate it
- tie every manuscript claim to a named audited artifact
- keep prompts, payoff matrices, and build steps transparent enough for a
  reviewer to audit quickly
