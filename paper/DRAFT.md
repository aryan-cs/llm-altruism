# Can LLM Agents Sustain a Society?
# Working Draft Notes

## Draft Thesis

The paper is no longer centered on classical games.

The central question is:

- if we place many LLM agents in a shared world with scarce resources, can they
  create and maintain a self-sustaining society?

Repeated games remain in the paper as precursor diagnostics for:

- baseline instability
- prompt steerability
- benchmark recognition
- explicit game prompts versus indirect fiction narratives with the same incentive structure

## Main Empirical Story

1. In the corrected scarcity society, `task-only` produces the best final
   survival.
2. `Cooperative` produces more trade and alliances, but not the best survival.
3. Public reputation equalizes survival while preserving large differences in
   social organization.
4. The repeated-game precursor probes explain why these macro claims need
   caution: "baseline" behavior is prompt-sensitive and benchmark-sensitive.

## Writing Priority

The abstract, introduction, figures, and results ordering should all answer the
society question first. Part 1 should appear as a measurement and
interpretation layer, not as the final paper takeaway.
